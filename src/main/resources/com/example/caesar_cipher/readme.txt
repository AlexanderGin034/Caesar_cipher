Поля:
	- поле "Путь к файлу" содержит путь к файлу, с которого необходимо прочитать текст;
	- поле "Ключ шифрования/дешифрования" принимает целые числа больше нуля;
	- полу "Дополнительный текст" принимает текст-шаблон, по которому будет проводиться статистический анализ (если флажок активен, то текст из загруженного файла будет считываться не в поле "Текст шифрования/дешифрования", а в поле - "Дополнительны текст);
	- поле "Результат шифрования/дешифрования" принимает результат.
Кнопки:
	- кнопка "Ракировка" перемещает текст из поля "Результата шифрования/дешифрования" в "поле Текст шифрования/дешифрования", удаляя текст из поля "Результат шифрования/дешифрования".
